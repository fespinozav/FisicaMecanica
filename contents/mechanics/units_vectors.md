# Magnitudes, Unidades y Vectores

## Objetivos de aprendizaje

- Usar correctamente el Sistema Internacional de unidades.
- Aplicar conversiones de unidades y analisis dimensional.
- Representar y operar vectores en dos y tres dimensiones.
- Descomponer magnitudes fisicas en componentes cartesianas.

## Magnitudes fundamentales

Las magnitudes fundamentales mas usadas en mecanica son longitud, masa y tiempo. A partir de ellas se construyen magnitudes derivadas como velocidad, aceleracion, fuerza y energia.

```{math}
[v] = L T^{-1}, \qquad [a] = L T^{-2}, \qquad [F] = M L T^{-2}
```

## Vectores

Un vector puede representarse por su magnitud y direccion o por sus componentes:

```{math}
\vec{A} = A_x \hat{i} + A_y \hat{j} + A_z \hat{k}
```

```{math}
|\vec{A}| = \sqrt{A_x^2 + A_y^2 + A_z^2}
```

## Habilidades clave

- Sumar vectores por componentes.
- Calcular modulo y direccion.
- Interpretar el producto escalar y el producto vectorial.
- Traducir diagramas geometricos a expresiones algebraicas.
