# Cinematica en Una Dimension

La cinematica describe el movimiento sin estudiar sus causas. En una dimension, la posicion, la velocidad y la aceleracion se tratan como funciones escalares del tiempo con signo.

## Objetivos de aprendizaje

- Interpretar graficos de posicion, velocidad y aceleracion.
- Resolver movimientos rectilineos uniforme y uniformemente acelerado.
- Conectar descripcion grafica, algebraica y verbal del movimiento.

## Ecuaciones basicas

```{math}
v(t) = \frac{dx}{dt}, \qquad a(t) = \frac{dv}{dt}
```

Para aceleracion constante:

```{math}
x(t) = x_0 + v_0 t + \frac{1}{2} a t^2
```

```{math}
v(t) = v_0 + a t
```

```{math}
v^2 = v_0^2 + 2 a (x - x_0)
```

## Ideas clave

- La pendiente de un grafico `x` versus `t` entrega velocidad.
- El area bajo un grafico `v` versus `t` entrega desplazamiento.
- Un signo negativo no implica rapidez negativa, sino direccion opuesta al eje elegido.

## Problemas tipo

1. Caida libre con y sin velocidad inicial.
1. Frenado de un vehiculo con aceleracion constante.
1. Encuentro entre dos moviles en linea recta.
