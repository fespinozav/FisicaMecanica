# Rotacion de Cuerpos Rigidos

La dinamica rotacional extiende muchas ideas de la traslacion: posicion angular, velocidad angular, aceleracion angular, torque y momento de inercia.

## Objetivos de aprendizaje

- Describir rotacion alrededor de un eje fijo.
- Calcular torque de una fuerza.
- Aplicar la segunda ley de Newton para rotacion.
- Usar energia y momentum angular en sistemas rigidos.

## Cinematica angular

```{math}
\omega = \frac{d\theta}{dt}, \qquad \alpha = \frac{d\omega}{dt}
```

Para aceleracion angular constante:

```{math}
\theta = \theta_0 + \omega_0 t + \frac{1}{2}\alpha t^2
```

## Dinamica rotacional

```{math}
\tau = \vec{r} \times \vec{F}
```

```{math}
\sum \tau = I \alpha
```

```{math}
K_{\mathrm{rot}} = \frac{1}{2} I \omega^2
```

## Problemas tipo

1. Poleas con masa.
1. Rodadura sin deslizamiento.
1. Barras y discos sometidos a torques externos.
