# Momentum Lineal y Colisiones

El momentum lineal es especialmente util cuando las interacciones ocurren en intervalos cortos o cuando las fuerzas externas netas son despreciables.

## Objetivos de aprendizaje

- Aplicar conservacion del momentum lineal.
- Relacionar impulso con cambio de momentum.
- Analizar choques elastico, inelastico y completamente inelastico.

## Ecuaciones esenciales

```{math}
\vec{p} = m \vec{v}
```

```{math}
\vec{J} = \int \vec{F} \, dt = \Delta \vec{p}
```

Si la fuerza externa neta es nula:

```{math}
\sum \vec{p}_i = \sum \vec{p}_f
```

## Ideas clave

- La energia cinetica no siempre se conserva en una colision.
- El momentum total del sistema si se conserva cuando el sistema esta aislado.
- En dos dimensiones, la conservacion se aplica por componentes.

## Problemas tipo

1. Retroceso.
1. Choques entre carros o particulas.
1. Explosiones idealizadas.
