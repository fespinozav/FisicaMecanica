# Cinematica en Dos Dimensiones

En dos dimensiones el movimiento se estudia componente por componente. Esto permite describir trayectorias planas como el movimiento parabólico o el movimiento circular.

## Objetivos de aprendizaje

- Describir movimiento plano usando vectores de posicion, velocidad y aceleracion.
- Resolver problemas de proyectiles.
- Relacionar componentes independientes con la trayectoria total.

## Formulacion vectorial

```{math}
\vec{r}(t) = x(t)\hat{i} + y(t)\hat{j}
```

```{math}
\vec{v}(t) = \frac{d\vec{r}}{dt}, \qquad \vec{a}(t) = \frac{d\vec{v}}{dt}
```

## Movimiento parabolico

Sin resistencia del aire:

```{math}
x(t) = x_0 + v_0 \cos\theta \, t
```

```{math}
y(t) = y_0 + v_0 \sin\theta \, t - \frac{1}{2} g t^2
```

## Extensiones utiles

- Movimiento circular uniforme.
- Aceleracion tangencial y normal.
- Trayectorias observadas desde distintos marcos de referencia.

## Problemas tipo

1. Alcance y altura maxima de un proyectil.
1. Tiempo de vuelo de un objeto lanzado desde una plataforma.
1. Analisis de velocidad relativa en dos ejes.
